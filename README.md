# Rock, Paper, Scissors Game

Let's play a classic game of Rock, Paper, Scissors!

To play, simply choose one of the following options:

- 🪨 Rock
- 📄 Paper
- ✂️ Scissors

### Rules

- Rock crushes scissors.
- Scissors cuts paper.
- Paper covers rock.

Make your choice and see if you can beat the computer!
